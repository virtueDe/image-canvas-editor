## 6. 代码规范提醒

### TypeScript
- 使用严格类型
- 避免 `any`
- `textOverlay` 字段默认可空，避免人为构造哨兵对象

### Vue
- UI 层只调用桥接动作，不直接操作 core 状态结构
- 保持现有 `script setup` 与 `computed` 风格

### 测试
- 优先补 `editor/core/src/*.test.ts`
- 先锁历史与兼容，再做 UI 联调
