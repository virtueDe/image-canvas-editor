# 文档维护细则

## AGENTS.md 只保留

- 关键路径、入口、工作区结构
- 必需命令
- 分层边界与禁止事项
- 提交前必做验证
- 提交与 PR 格式

## 移出 AGENTS.md

- 解释“为什么”
- 长示例、代码块、重复规则
- 可从源码或 README 直接检索的信息
- 偶发流程、维护策略、压缩方法细节

## 手动冒烟

- 上传图片
- 左右旋转、水平/垂直翻转
- 进入、取消、应用裁剪
- 预设滤镜与参数调节
- 保存与恢复草稿
- 导出 PNG

## 何时更新文档

- workspace 结构、包名、命令变更
- `dist/` 输出路径变更
- `editor/core` 与 `editor/vue3` 边界变更
- 草稿、导出、状态结构兼容性变更

## 压缩守则

- AGENTS 目标是高密度索引，不是教程
- 新增规则前先合并同类项
- 路径优先于解释，事实优先于口号
- 行数失控时回看 `../../CLAUDE.md-压缩指南.md`
